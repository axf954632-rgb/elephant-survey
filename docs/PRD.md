# 大象问卷 — 产品需求文档（PRD）

## 1. 项目概述

| 项目 | 说明 |
|------|------|
| **名称** | 个人能力12维度评估表（大象问卷） |
| **定位** | 个人成长/能力自评工具 |
| **目标用户** | 希望系统评估自身能力的个人用户（预计约100人使用） |
| **核心功能** | 36题自评 → 12维度雷达图 → 结果解读 → 分享海报 |
| **数据存储** | 浏览器 localStorage（单机版，无后端数据库） |
| **部署方式** | Vercel（主）+ 腾讯云 COS（备选） |

## 2. 信息架构

### 2.1 12个评估维度

| 页码 | 大维度 | 细维度 | 题号 |
|:--:|:--|:--|:--|
| P1 | 驱动与修身 | 敏锐学习 | Q1–Q3 |
| P2 | 驱动与修身 | 自律自洽 | Q4–Q6 |
| P3 | 驱动与修身 | 情绪韧性 | Q7–Q9 |
| P4 | 思维与策略 | 逻辑思辨 | Q10–Q12 |
| P5 | 思维与策略 | 系统思考 | Q13–Q15 |
| P6 | 思维与策略 | 决策判断 | Q16–Q18 |
| P7 | 执行与产出 | 执行效率 | Q19–Q21 |
| P8 | 执行与产出 | 业务能力 | Q22–Q24 |
| P9 | 执行与产出 | 复盘改进 | Q25–Q27 |
| P10 | 人际与管理 | 沟通情商 | Q28–Q30 |
| P11 | 人际与管理 | 上下协作 | Q31–Q33 |
| P12 | 人际与管理 | 领导管理 | Q34–Q36 |

### 2.2 评分规则

- 每题 1–5 分（1=最差，5=最好）
- 每个细维度满分：3题 × 5分 = **15分**
- 总分满分：12维度 × 15分 = **180分**

## 3. 页面流程

```
首页
  ↓ 点击「开始测评」
开始说明页
  ↓ 点击「开始测评」
问答页（共12页，每页3题）
  ↓ 全部答完
结果页（雷达图 + 详细得分）
  ↓ 点击「生成分享海报」
海报预览弹窗（可保存/转发）
```

## 4. 各页面需求

### 4.1 首页

- 左侧文字，右侧大象插画
- 标题：`个人能力`（蓝色）+ `12维度评估表`（深色）
- 小字：`创始人：醉客`
- 按钮：新拟态「开始测评」按钮
- 响应式：PC 左右分栏，移动端上下堆叠

### 4.2 开始说明页

- 标题：开始说明
- 文案：
  - 本网站旨在帮助用户评估自身能力水平，为用户的能力成长指引方向。
  - 请注意由于自我评估的主观性，不同人的评估结果未必具有比较价值，请注意甄别。
  - 题目中 1 为最差，5 为最好。
- 按钮：新拟态「开始测评」按钮

### 4.3 问答页

- 每页显示 1 个维度（3 道题）
- 顶部进度条（当前维度 / 12）
- 维度标题 + 所属大维度标签
- 每道题 5 个圆形评分按钮（1-5分）
- 第3题、第6题选中后显示对应描述文字
- 底部「上一页」「下一页」导航（第一页无上一页）
- 键盘快捷键：1-5 评分，Enter 翻页
- 自动保存进度到 localStorage

### 4.4 结果页

- 总分显示
- 十二维度雷达图
- 四大维度概览卡片
- 十二维度详解列表（含等级标签、进度条、文字解读）
- 「生成分享海报」按钮
- 「重新测评」按钮

### 4.5 分享海报

- 宽度 320px，适合移动端保存/转发
- 包含：标题、综合得分、12维度分数、二维码
- 二维码指向当前网站地址
- 预览弹窗提示：长按图片保存 / 转发到朋友圈
