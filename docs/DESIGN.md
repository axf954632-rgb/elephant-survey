# 大象问卷 — 设计规范文档

## 1. 设计方向

**风格名称：** 软糖新拟态（Soft Candy Neumorphism）

将柔和的新拟态质感与卡通插画的活泼感结合，既满足用户要求的凸起按钮效果，又与大象 IP 形象和谐共存。

## 4. 新拟态组件规范

### 4.1 凸起按钮（Primary Button）

```css
.neu-btn {
  background: linear-gradient(145deg, #f5f8fc, #e5eaf2);
  box-shadow:
    8px 8px 16px #d1d9e6,
    -8px -8px 16px #ffffff;
  border-radius: 16px;
  padding: 16px 40px;
  font-size: 18px;
  font-weight: 700;
  color: #2C3E50;
}

.neu-btn:hover {
  box-shadow:
    12px 12px 24px #d1d9e6,
    -12px -12px 24px #ffffff;
  transform: translateY(-2px);
}

.neu-btn:active {
  background: linear-gradient(145deg, #e5eaf2, #f5f8fc);
  box-shadow:
    inset 4px 4px 8px #d1d9e6,
    inset -4px -4px 8px #ffffff;
}
```

### 4.2 新拟态卡片

```css
.neu-card {
  background: linear-gradient(145deg, #f5f8fc, #e5eaf2);
  box-shadow:
    12px 12px 24px #d1d9e6,
    -12px -12px 24px #ffffff;
  border-radius: 24px;
}
```

### 4.3 内凹容器

```css
.neu-inset {
  background: #EDF2F8;
  box-shadow:
    inset 6px 6px 12px #d1d9e6,
    inset -6px -6px 12px #ffffff;
  border-radius: 20px;
}
```

## 5. 页面布局规范

### 5.1 首页

- PC 端：左右分栏，左文右图
- 移动端：上下堆叠，图在上文在下
- 最大内容宽度：max-w-6xl
- 大象图尺寸：移动端 224px，平板 288px，PC 416px
- 大象图动画：上下浮动 `float 3s ease-in-out infinite`

### 5.2 问答页

- 顶部固定进度条
- 内容居中，最大宽度：max-w-xl
- 每道题使用新拟态卡片
- 底部固定导航按钮

### 5.3 结果页

- 顶部标题居中
- 雷达图卡片居中
- 四大维度使用 2×2 网格
- 十二维度使用列表卡片
- 底部按钮并排（PC）或堆叠（移动端）

## 6. 评分组件规范

### 6.1 评分按钮

- 5 个圆形按钮，1-5 分
- 未选中：新拟态凸起，灰色文字
- 选中：大象蓝填充，白色文字，内凹阴影，放大 10%
- 按钮尺寸：44-48px

### 6.2 选中后描述

- 位置：评分按钮下方
- 颜色：灰色（`text-gray-500`）
- 字体：12px
- 普通题显示：很低/较低/一般/较高/很好
- 特殊题（第3、6题）显示：对应分数的详细描述

## 7. 响应式断点

| 断点 | 宽度 | 设备 | 布局调整 |
|:--|:--|:--|:--|
| sm | ≥640px | 大屏手机/小平板 | 按钮并排，标题放大 |
| md | ≥768px | 平板 | 首页左右分栏 |
| lg | ≥1024px | PC | 内容最大宽度限制 |

## 8. 插画与图像

- 大象插画：透明背景 PNG
- 动画：上下浮动 ±12px，3s 循环
- 投影：淡蓝色投影 `drop-shadow-[0_20px_40px_rgba(74,144,226,0.15)]`

## 9. 动效规范

| 元素 | 动效 | 时长 | 缓动 |
|:--|:--|:--|:--|
| 大象图 | 上下浮动 | 3s | ease-in-out |
| 按钮 hover | 阴影放大 + 上移 | 0.2s | ease |
| 按钮 active | 内凹 + 恢复位置 | 0.2s | ease |
| 进度条 | 宽度变化 | 0.5s | ease-out |
| 评分按钮选中 | 缩放 + 颜色变化 | 0.2s | ease |

## 10. 海报设计规范

- 宽度：320px
- 背景：#EDF2F8
- 圆角：24px
- 内容：标题 + 总分 + 12维度分数 + 二维码 + 底部文案
- 二维码尺寸：80×80px
- 不使用 SVG 图表，使用纯 DOM 布局
